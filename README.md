# Task Manager CLI

A command-line task management application built with Python.

## Features

- ✅ Add new tasks
- ✅ View all tasks (organized by status)
- ✅ Mark tasks as complete
- ✅ Delete tasks
- ✅ Persistent storage (tasks saved to JSON file)
- ✅ User-friendly interface with error handling

## How to Run

1. Make sure you have Python 3 installed
2. Navigate to the project directory
3. Run: `python3 task_manager.py`

## Technical Details

- **Language**: Python 3
- **Storage**: JSON file (`tasks.json`)
- **Data Structure**: List of dictionaries with task description and completion status

## Learning Goals

- Python fundamentals (lists, dictionaries, loops, conditionals)
- File I/O and JSON handling
- Error handling with try/except
- User input validation
- Git version control
- Writing clean, documented code
