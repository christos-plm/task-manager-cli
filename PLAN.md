# Task Manager CLI - Project Plan

## Project Goal
The program should create a to-do list for the user

## Features - Week 1 (Basic Working Version)
- [ ] Display a menu with options
- [ ] Add a new task (just description)
- [ ] View all tasks in a list
- [ ] Exit the program
- [ ] Menu loops until user exits

## Features - Week 2 (Enhanced Version)
- [ ] Assign ID numbers to tasks
- [ ] Delete a task by ID
- [ ] Mark a task as complete/incomplete
- [ ] Save tasks to JSON file
- [ ] Load tasks from JSON file when program starts

## Features - Week 3 (Object-Oriented Version)
- [ ] Create a Task class
- [ ] Each task has: id, description, completed status, created_date
- [ ] Refactor code to use Task objects

## Features - Week 4 (Polish & Bonus)
- [ ] Add due dates to tasks
- [ ] Add priority levels (low, medium, high)
- [ ] Improve display formatting
- [ ] Add error handling (invalid inputs)
- [ ] Write a professional README
